My Projects

Hardware-Software Codesign of DNN Accelerators Using Approximate Posit Multiplierse

Designed a hardware accelerator for neural networks (Simba) by improving the design of the MAC unit using Verilog and changing the number system of the chip to a modified Posit number system. Achieved 2.2 speedup, 3.1 lower energy use, and 3.2 smaller area without accuracy loss compared to the baseline using the Timeloop accelerator simulator.

KALAM: toolKit for Automating high-Level synthesis of Analog computing systeMs

Developed a tool using Python for generating and simulating analog computing systems using margin propagation (MP)-based circuits. The tool supports generation of the following circuits: Bayesian networks using soft MP based analog gates, LDPC decoders (Minsum), and ANN inference via MP-based MAC circuits.

A Novel SRAM-Based Cache Architecture for Enhanced Machine Learning Workloads

Designed a near memory (SRAM) accelerator for DNN workloads taking advantage of approximate Posit MAC units and integrating tailored vector Multiply-Accumulate (MAC) Units within SRAM sub-arrays. W analyzed the performance of the proposed design using the Timeloop accelerator simulator as well as the performance of the MAC units made using Verilog.

View Report

Enhancements to DRAM On-Die ECC for Emerging Integrity Issues

Simulated the reliability of different on-die ECC schemes for DRAM, such as SECDED codes, Reed Solomon codes,BCH codes, and Chipkill using the DRAM-Faultsim reliability simulator using C++. Analyzed the encoder/decoder circuit for SECDED, BCH and Reed Solomon codes in order to find the optimum on die ECC scheme in terms of reliability, performance and storage overhead.
